🚀 AutoDiag — OBD2 Diagnostic + Arduino Display
AutoDiag — мобильное приложение для Android, которое подключается к автомобилю через Bluetooth OBD2 адаптер (ELM327) и показывает параметры двигателя в реальном времени. Дополнительно можно подключить внешний дисплей на Arduino Nano + HC-05 + LCD 20×4 для дублирования температуры и расхода топлива.

🔥 Возможности

-Подключение к OBD2 адаптеру по Bluetooth

-Отображение скорости, оборотов, температуры, нагрузки, MAP

-Расчёт расхода топлива через MAF-датчик

-Графики в реальном времени

-Сохранение поездок в Room

-Экспорт отчётов в PDF

-Чтение и очистка DTC ошибок (Mode 03/04)

-Профили автомобилей (Toyota, Subaru, Generic)  

-Внешний дисплей на Arduino (HC-05 + LCD 20×4)

-Тёмная тема Material Design

📱 Минимальные требования

Android	5.0 (API 21) и выше,
Bluetooth	4.0+ с поддержкой SPP,
Автомобиль	OBD2 (1996+),
Адаптер	ELM327 (Bluetooth).

📦 Установка

Скачай APK из релиза или собери из исходников:

bash
git clone https://github.com/your-username/autodiag.git

Открой проект в Android Studio.

Собери и установи на смартфон.

🔌 Подключение к автомобилю

Вставь OBD2 адаптер в диагностический разъём (под рулевой колонкой).

Включи зажигание.

В настройках Bluetooth телефона сопрягись с адаптером (PIN: 1234 или 0000).

Запусти приложение → выбери профиль авто → нажми «Подключиться».

🧩 Внешний дисплей на Arduino
Прошивка для Arduino — в папке arduino/autodiag_display.ino.

Формат данных
Приложение отправляет строку:
T:95;F:8.5\n
где T — температура (°C), F — расход (л/100км).

На LCD отображается:
text
Temp: 95 C
Fuel: 8.5 l/100km
